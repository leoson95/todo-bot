# 📝 TodoBot - ربات مدیریت کارها و خریدها

ربات تلگرام شخصی و تمیز برای مدیریت کارها، خریدها و یادآوری‌های زمان‌بندی شده.

## ✨ ویژگی‌ها

- 📌 **پیام ثابت و تمیز**: چت همیشه مرتب و بدون شلوغی می‌ماند.
- 📂 **۶ دسته‌بندی اختصاصی**: فوری، کارای سالن، خریدای سالن، کارای خونه، خریدای خونه، کارای شخصی.
- ⏰ **یادآوری زمان‌بندی شده**: تنظیم ساعت و دوره تکرار (هر روز، یک روز در میان، هفتگی).
- ✅ **دکمه‌های هوشمند**: دکمه `انجام شد` و `دیدم` برای پیام‌های یادآوری.
- 🔄 **حذف خودکار پیام‌های موقتی**: چت همیشه تمیز و مرتب می‌ماند.

## 🚀 نحوه استفاده

1. به ربات `/start` بزن.
2. هر متنی بفرستی به عنوان کار جدید ثبت می‌شود.
3. دسته‌بندی را انتخاب کن.
4. اگر می‌خواهی یادآوری بذاری، ساعت و دوره تکرار را تنظیم کن.

## 🔧 فناوری‌ها

- **PHP 8+** + Webhook
- **SQLite** (ذخیره‌سازی سبک و سریع)
- بدون فریم‌ورک سنگین
- بهینه شده برای **Railway Free Tier**

## 🚀 دیپلوی روی Railway

1. ریپو را Fork کن.
2. در Railway یک پروژه جدید ایجاد کن.
3. ریپو را اضافه کن.
4. متغیر محیطی `BOT_TOKEN` را تنظیم کن (اختیاری - توکن در کد هم قرار گرفته).
5. فایل `cron.php` را با سرویس **[cron-job.org](https://cron-job.org)** هر ۱ دقیقه اجرا کن.
